# Ghostty Config

[Ghostty](https://ghostty.org/) の設定ファイル。

## セットアップ

```bash
git clone git@github.com:pomesaka/config-ghostty.git ~/.config/ghostty
```

## 設定内容

- **テーマ**: Gruvbox Dark
- **Quick Terminal**: `Ctrl+Cmd+Space` でトグル
- **ペイン移動**: `Ctrl+Cmd+h/j/k/l`
- **改行入力**: `Shift+Enter`
- **タイトルバー**: 非表示

## 設定の再読み込み

`Cmd+Shift+,`
